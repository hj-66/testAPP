# Changelog

## [1.2.0] - 2026-04-30

### Added

- 메모 본문 글꼴 선택 기능 추가

### Changed

- 글꼴 변경이 일반 메모 입력창에도 즉시 적용되도록 수정
- 한글 본문에도 글꼴 변경이 적용되도록 한글 지원 시스템 글꼴을 사용
