# Kotlin Memo App

## 프로젝트 개요

이 프로젝트는 Kotlin으로 개발하는 macOS용 메모장 앱입니다.  
기본적인 메모 작성, 수정, 삭제 기능을 시작으로 이후 다양한 기능을 확장할 수 있도록 설계합니다.

## 개발 목표

- macOS에서 실행 가능한 데스크톱 메모장 앱 제작
- Kotlin 기반 GUI 애플리케이션 개발 경험 쌓기
- 기능을 점진적으로 추가할 수 있는 구조 설계
- 코드 유지보수성과 확장성을 고려한 프로젝트 구성

## 기술 스택

- Language: Kotlin
- Platform: macOS
- UI Framework: Kotlin Compose Desktop
- Build Tool: Gradle
- IDE: IntelliJ IDEA 권장

## 기본 기능

### 1. 메모 작성

사용자가 새로운 메모를 작성할 수 있다.

### 2. 메모 목록 조회

저장된 메모 목록을 확인할 수 있다.

### 3. 메모 상세 보기

선택한 메모의 내용을 확인할 수 있다.

### 4. 메모 수정

기존 메모의 제목과 내용을 수정할 수 있다.

### 5. 메모 삭제

필요 없는 메모를 삭제할 수 있다.

## 주요 기능 (구현 기준)

### 1. 메모 관리

- 메모한 내용은 .txt 파일로 실제 내 컴퓨터에 저장 가능
- 파일에서 .txt 파일을 불러와 편집 가능

---

### 2. 메모 검색 기능

- 제목 및 내용 기반 키워드 검색
- 실시간 검색 (입력 시 필터링)
- 대소문자 구분 없이 검색 지원

---

### 5. 저장 기능

- 메모 작성 중 command+s를 통해 실제 저장

---

### 6. 다크 모드

- 라이트 모드 / 다크 모드 전환 지원
- 시스템 설정 연동 (macOS 테마 감지)
- 사용자 설정 저장

---

### 7. 단축키 지원

- 새 메모: `Cmd + N`
- 저장: `Cmd + S`
- 삭제: `Cmd + Delete`
- 검색: `Cmd + F`
- 단축키 커스터마이징 가능 (확장 고려)

---

### 8. 로컬 파일 저장

- JSON 또는 텍스트 파일 기반 저장
- 앱 실행 시 데이터 자동 로드
- 파일 경로 설정 가능

---

### 9. SQLite 연동

- 메모 데이터를 SQLite DB에 저장
- 대량 데이터 처리 및 검색 성능 향상
- DAO 패턴 적용

---

### 10. 마크다운 지원

- 메모 내용에 Markdown 문법 지원
- 실시간 미리보기 기능
- 마크다운 적용/비적용 버튼을 통해 실시간으로 확인 가능하게 변경
- 주요 문법:
  - 제목 (#)
  - 리스트 (-)
  - 코드 블록 (```)
  - 강조 (**bold**, *italic*)

---

### 11. 메모 정렬 기능

- 생성일 / 수정일 / 제목 기준 정렬
- 오름차순 / 내림차순 선택 가능

---

### 12. 삭제 기능

- 삭제 시 목록에서 즉시 제거
- 삭제된 메모는 별도 휴지통으로 이동하지 않음

---

### 13. 백업 및 복원 기능

- 수동 백업 (파일 export)
- 자동 백업 (주기 설정 가능)
- 백업 파일로부터 데이터 복원

---

### 14. 글꼴 편집 기능

- 굵게, 밑줄, 기울임 등 글꼴 강조기능
- 글씨 색깔 변경 기능 추가

---

## 시스템 구조 (확장 고려)

### 아키텍처

- MVVM 또는 MVC 패턴 적용
- UI / 비즈니스 로직 / 데이터 계층 분리

---

### 계층 구조

```text
ui/            → 화면 (Compose / JavaFX)
viewmodel/     → 상태 관리 및 로직 연결
service/       → 비즈니스 로직
repository/    → 데이터 접근
database/      → SQLite / 파일 저장
model/         → 데이터 구조
```

---

## UI 설계 프롬프트

Kotlin Compose Desktop을 사용하여 macOS 스타일의 메모장 UI를 구현한다.

### 1. 전체 레이아웃

- 좌측: 메모 리스트 패널
- 우측: 메모 상세 편집 패널

```text
-----------------------------------------
| 메모 리스트 |      메모 내용          |
|             |                         |
|             |                         |
|             |                         |
-----------------------------------------
```

## 2. 메모 리스트 영역 (좌측 패널)

### 구성 요소

- 상단 파일에서 불러오기 창
- 저장버튼

### 메모 아이템 UI

- 제목
- 미리보기 (한 줄)
- 생성 날짜
- 즐겨찾기 아이콘

### 동작

- 클릭 시 우측 패널에 메모 내용 표시
- 검색 입력 시 리스트 필터링
- 즐겨찾기 아이콘 클릭 시 상태 토글

---

## 3. 메모 상세 영역 (우측 패널)

### 구성 요소

- 제목 입력 필드
- 즐겨찾기 버튼
- 메모 내용 텍스트 에디터
- 저장 버튼
- 다른 이름으로 저장 버튼
- 삭제 버튼

### 동작

- 입력 시 자동 저장
- 수정 시 `updatedAt` 갱신
- 삭제 시 목록에서 즉시 제거

---

## 4. 상태 관리

다음 상태를 관리해야 한다:

- 현재 선택된 메모
- 메모 리스트
- 검색 키워드
- 다크 모드 여부
- 정렬 기준
- 선택된 메모의 `.txt` 파일 경로

---

## 5. UI 스타일

- macOS 느낌의 미니멀 디자인
- 라운드 처리된 카드 UI
- 충분한 여백 사용
- 다크 모드 지원

### 색상

- 라이트 모드: 흰색 기반
- 다크 모드: 어두운 회색 기반

---

## 6. 사용자 경험 (UX)

- 클릭 최소화 (빠른 메모 작성)
- 키보드 중심 사용 가능
- Compose recomposition을 활용한 부드러운 상태 전환

---

## 7. 추가 UI 요소 (확장 고려)

- 설정 화면
- 마크다운 미리보기 패널 (Split View)
